# Real-Time E-Commerce ETL Pipeline

## 📌 Project Overview
This project implements a scalable ETL pipeline using Python and AWS.

## 🚀 Features
- Extract data from AWS S3
- Transform data using Pandas
- Load into AWS RDS
- Workflow automation using Apache Airflow
- Data validation and logging

## 🛠 Tech Stack
- Python
- Pandas
- AWS S3
- AWS RDS
- Apache Airflow
- Docker
